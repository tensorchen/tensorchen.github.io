Python爬虫框架爬取美团所有餐馆和菜品数据

  • python
  • scrapy
  • spider
computer

网络爬虫（又被称为网页蜘蛛，网络机器人），是一种按照一定的规则，自动的抓取万维网信息的程序或者脚本。

本实例使用的是 Python 爬虫框架 Scrapy 爬取美团官网在成都市内的所有餐馆以及菜品信息。

Scrapy 安装

Scrapy 作为常用的 Python 库，可以很方便的使用 pip ——Python 包管理工具安装。

$ pip install Scrapy

定义数据格式

定义需要爬取数据的数据类型与格式，编辑 items.py 文件
数据类型均为 Field，但 Field 可接受任何 python 类型
如果需要定义美团餐馆 json 数据格式如下

{
    "r_link":                           #餐馆链接
    "r_name":                           #餐馆名称
    "r_foods":                          #餐馆菜品列表
        [
         {
           "f_name":                    #菜品名称
           "f_price":                   #菜品价格
         }
        ]
}

对应代码如下：
from scrapy.item import Item, Field

class RrsItem(Item):
    r_name = Field()
    r_link = Field()
    r_foods = Field()

class FoodItem(Item):
    f_name = Field()
    f_price = Field()

主要解决如下常遇问题：
1.如何从复杂的 html代码中定位你所需要的数据？
使用 CSS选择器 或者 XPATH

2.如何进入详细页面提取数据，或者数据分散于不同页面？
可参见下面代码实现和注释

3.如何处理数据分页，爬取下一页数据？
可参见下面代码实现和注释

import scrapy

from rrs.items import RrsItem, FoodItem

class MeituanSpider(scrapy.Spider):
    name = "meituan"
    allowed_domains = ["cd.meituan.com"] 
    start_urls = [
        'http://cd.meituan.com/category/meishi/'
    ]

    def parse(self, response):
        #通过CSS选择器，XPATH，定位餐馆数据所在标签，提取相关信息
        for sel in response.css("div.basic.cf > a"):        
            rrs_item = RrsItem()
            rrs_item['r_name'] = sel.xpath('text()').extract()
            rrs_item['r_link'] = sel.xpath('@href').extract()

            #并将具体餐馆的链接传递给 parse_food 函数,以便提取详细的菜品信息    
            yield scrapy.Request(rrs_item['r_link'][0], meta = {'rrs_item': rrs_item},
                    callback = self.parse_food)

        #提取下一页链接，进入下一页并提取下一页数据
        next_page = response.css("li.next > a::attr('href')")

        if next_page:
            url = response.urljoin(next_page[0].extract())
            yield scrapy.Request(url, self.parse)

    #提取餐馆详细的菜品数据
    def parse_food(self, response):

        rrs_item = response.meta['rrs_item']

        r_foods = []

        for sel in response.css("div.menu__items > table > tr > td"):
            r_food = FoodItem()
            r_food['f_name'] = sel.xpath('text()').extract()
            r_food['f_price'] = sel.xpath('span/text()').extract()

            r_foods.append(r_food)

        rrs_item['r_foods'] = r_foods

        yield rrs_item

保存数据

如下在项目根目录执行命令则存储 json 格式数据于 rrs_all_page.json 文件

$ scrapy crawl meituan -o rrs_all_page.json
